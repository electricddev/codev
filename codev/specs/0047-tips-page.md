# Spec 0047: Tips & Tricks Page

## Overview

Create a tips and tricks documentation page for Codev users, linked from the main README.

## Requirements

1. Create `docs/tips.md` with practical tips covering:
   - Permission management (`--dangerously-skip-permissions`)
   - Recovery & resumption (`/resume` command)
   - Agent Farm dashboard features (hammer icon, `af consult`)
   - Productivity tips (parallel consultations, quick spawning)
   - Git workflow tips
   - Debugging commands
   - Tower usage

2. Add link to tips page in README.md under "Learn about Codev" section

## Acceptance Criteria

- [ ] `docs/tips.md` exists with useful tips
- [ ] README.md links to the tips page
- [ ] Tips are accurate and actionable
